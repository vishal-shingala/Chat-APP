# 🗨️ Chat APP

A real-time chat application built using **Node.js**, **Express**, and **Socket.io**. This app allows users to chat in real time with online presence detection, secure authentication, and a clean UI.

## 🚀 Features

- ✅ Real-time messaging using **Socket.io**
- ✅ User authentication and session handling
- ✅ Online/offline user presence tracking
- ✅ Responsive and interactive UI
- ✅ Firebase (or MongoDB) for data storage
- ✅ Modular and organized codebase

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Real-time:** Socket.io
- **Database:** MongoDB
